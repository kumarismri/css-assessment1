What is CSS?

  1. CSS stands for Cascading Style Sheets.
  2. CSS describes how HTML elements are ti be displayed on screen, paoer, paper, or in othere media
  3. Css is not necessaryCSS saves a lot of work. It can controll the layout of multiple web pages all at once
  4. External stylesheets are stored in CSS files.

CSS can be added to HTML documents in 3 ways:

  • Inline - by using the style attributes inside HTML elements
  • Internal - by using a style element in the head section
  • External - by using a link element to link to an external CSS file
  • The most common way to add CSS. is to keep the styles in external CSS files. However. In this
    tutorial we will use inline and internal styles,    because this is easier to demonstrate, and easier
    for you to try it yourself

When comparing CSS class vs ID, difference is that CSS applies astyle to multiple elements. ID, on the other hand,applies a style to one unique element. ID is also
special in that you can use a special URL to link directly to an elements and it's used by JavaScript.

In CSS, selectors are used to target a specific element or range of element or range of elements on a web page. Once an element has been targeted a style or set of styles can be applied to the
element.

There is a wide range of selectors available. Two of the most commonly used are class and ID. Both are used to target elements to which a style should be applied.
